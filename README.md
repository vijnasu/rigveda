# ಕನ್ನಡದಲ್ಲಿ ವೈಧಿಕ ಸಾಹಿತ್ಯ

### ಕೃತಜ್ಞತೆ

ಕನ್ನಡದಲ್ಲಿ ಸಸ್ವರ ಋಗ್ವೇದ ಸಂಹಿತೆಯನ್ನು ಮೊತ್ತಮೊದಲು ಅಂತರ್ಜಾಲಕ್ಕೆ ಅಳವಡಿಸಿದ ಮಿತ್ರ ಶ್ರೀ ದಯಾನಂದ ಐತಾಳರಿಗೆ ವಂದನೆಗಳು.
